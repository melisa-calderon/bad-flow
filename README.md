# bad-flow
*This software retrieves album information from Spotify*

## Installation
* Runs in Python 2.7
* Run commands: 
  * sudo pip install spotipy
  * sudo pip install pandas
  * sudo pip install requests
  * sudo pip install spotipy
  * sudo pip install sqlalchemy
  * sudo pip install plotly
      
* To install
  * sudo install bad-flow
